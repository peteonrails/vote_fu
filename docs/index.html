Simple, expressive API

Everything you need, nothing you don't.

Setup your models 
class User < ApplicationRecord
  include VoteFu::Voter
end

class Post < ApplicationRecord
  include VoteFu::Voteable
end
Cast votes 
# Upvote / Downvote
user.vote_on(post, value: 1)
user.vote_on(post, value: -1)

# Star ratings
user.vote_on(post, value: 5)

# Scoped voting
user.vote_on(comment, value: 1, scope: :helpful)
user.vote_on(comment, value: 5, scope: :quality)
Query and rank 
# Built-in scopes
Post.most_voted
Post.highest_rated
Post.controversial

# Ranking algorithms
Post.by_wilson_score
Post.by_reddit_hot
Post.by_hacker_news_rank
ViewComponents 
<!-- Vote buttons -->
<%= render VoteFu::VoteWidgetComponent.new(
  voteable: @post,
  voter: current_user
) %>

<!-- Star rating -->
<%= render VoteFu::StarRatingComponent.new(
  voteable: @post,
  max_stars: 5
) %>
